Some Links I Like/Find Interesting

Code/Software Related:

codecogs.com

 This is an online LaTeX equation editor, allows creating of images based on the equation you provide.

toptal.com/developers/sorting-algorithms

 Provides a visual of how a sorting algorithm behaves, as well as a code example and properties for each algorithm shown,

bigocheatsheet.com

 The BigO cheat sheet shows time and space complexities of data structures and sorting algorithms. They even have a purchasable poster!

ideone.com

 An online compiler and debugger for a very many programming languages. Great for on the go if you have a less capable device, or to run a piece of code without having to install the compiler or open/use an IDE.

Udemy.com

 (The link is a referral link) Udemy offers online courses on... well tons of things. I'm particularly interested in the software engineering and programming courses, but there really is a lot of other options like photography, marketing, and music just to name a few.

Game Related:

wowhead.com

 A World of Warcraft site for lots of in game information, useful if you play World of Warcraft.

reddit.com/r/thedivision

 A reddit for The Division 1 and 2, very useful community resource.
